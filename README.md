# Blog de café

Este es mi tercer sitio web estático que trata sobre un blog de café, con un apartado de Nosotros, Cursos, Contacto y Entrada

## · Pagina principal

En la pagina principal encontrara información de los cursos con sus cupos y precios, ademas de entradas acerca de tipos de granos, recetas y beneficios del café

## · Nosotros

Información de quienes son los responsables del proyecto

## · Cursos

Se ofrece 3 cursos de sobre extracción de café, taller la tostar y moler granos, y cuatro recetas de café

## · Contacto

Donde se puede envía mensajes con correo al o los encargados del sitio web

# Tecnologías usadas para este proyecto

- HTML para la estructura de la pagina

- Css para dar estilo a la pagina

- Javascript en el plugin que descargue en "Modernizr" para imágenes webp el cual permite que están se muestren en caso de ser soportados por el navegador y si no es soportado usara la imagen de formato jpg
